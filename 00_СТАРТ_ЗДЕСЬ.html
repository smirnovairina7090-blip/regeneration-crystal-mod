Кристалл регенерации

Полный маршрут от скачанного проекта до работающего блока в TLauncher

Что вы скачали

Этот ZIP пока не является готовым модом. В нём лежат Java-код, модель, текстура и настройки сборки.

Проект ZIP→GitHub Actions→готовый JAR→TLauncher

TLauncher не открывает папку с исходниками. Он запускает уже собранный файл .jar.

1. Распакуйте архив

  1. Нажмите по ZIP правой кнопкой мыши.
  2. Выберите Извлечь всё.
  3. Откройте распакованную папку.
На экране должны быть:
.github, src, blockbench_source, build.gradle, gradle.properties.

Не загружайте ZIP в GitHub одним файлом. Нужны файлы и папки изнутри.

2. Создайте репозиторий GitHub

  1. Откройте GitHub и войдите.
  2. Нажмите New repository.
  3. Название: regeneration-crystal-mod.
  4. Выберите Public или Private.
  5. Не добавляйте README и .gitignore.
  6. Нажмите Create repository.
Что должно появиться: пустая страница нового репозитория.

3. Загрузите проект

  1. Нажмите uploading an existing file или Add file → Upload files.
  2. Перетащите всё содержимое распакованной папки.
  3. Дождитесь окончания загрузки.
  4. Нажмите Commit changes.
Обязательная проверка: откройте .github → workflows. Там должен быть build-mod.yml.

4. Получите готовый JAR

  1. Откройте вкладку Actions.
  2. Выберите Build Regeneration Crystal Mod.
  3. Если запуск не начался автоматически, нажмите Run workflow.
  4. Дождитесь зелёной галочки.
  5. Откройте готовую сборку и прокрутите вниз до Artifacts.
  6. Скачайте regeneration-crystal-mod-jar.
  7. Распакуйте скачанный ZIP.
Итог: файл примерно regeneration_crystal-1.0.0.jar.

Если галочка красная, откройте шаг Build mod и сохраните текст ошибки.

5. Создайте сборку TLauncher

  1. Запустите TLauncher.
  2. Нажмите TL MODS внизу справа.
  3. Нажмите Create.
  4. Название: Кристалл регенерации.
  5. Загрузчик: NeoForge.
  6. Minecraft: 1.21.1.
  7. Версию NeoForge оставьте автоматически выбранной для 1.21.1.
  8. Нажмите Create.
Нужна связка: Minecraft 1.21.1 + NeoForge 21.1.x.

6. Установите JAR в TLauncher

Самый понятный способ

  1. В TL MODS выберите созданную сборку.
  2. Нажмите кнопку с гаечным ключом и отвёрткой.
  3. Выберите Manual installation / Ручная установка.
  4. Выберите файл regeneration_crystal-1.0.0.jar.
  5. Нажмите Install / Установить.

Запасной способ

  1. Откройте папку выбранной сборки.
  2. Откройте или создайте папку mods.
  3. Скопируйте туда JAR.

Не выбирайте исходный ZIP, .bbmodel или PNG. Нужен только JAR.

7. Проверьте мод в игре

  1. Выберите сборку Кристалл регенерации.
  2. Нажмите Войти в игру.
  3. В главном меню откройте список модов и найдите Regeneration Crystal.
  4. Создайте творческий мир.
  5. Найдите Кристалл регенерации через поиск инвентаря.
  6. Поставьте его и нажмите правой кнопкой мыши.
Ожидаемый результат: кристалл светится и выдаёт регенерацию игрокам в радиусе 5 блоков на 10 секунд.

Если что-то не работает

  • Нет Actions: проверьте файл .github/workflows/build-mod.yml.
  • Красная сборка: откройте шаг Build mod и сохраните ошибку.
  • Мод не виден: проверьте 1.21.1, NeoForge и JAR в выбранной сборке.
  • Игра вылетает: сохраните файл logs/latest.log из папки сборки.
Запомните: проект ZIP → GitHub Actions → готовый JAR → TLauncher с NeoForge 1.21.1
